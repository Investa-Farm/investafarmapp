1. The Rainfall-Yield Relationship

Maize yield variability is strongly associated with total seasonal rainfall (R² = 0.44), with grain yield ranging from crop failure to 3.8 t/ha under total seasonal rainfall of 37–592mm. NDVI itself shows a significant positive relationship with rainfall (R² = 0.437), and NDVI is strongly driven by precipitation (correlation r = 0.72–0.88).

This means your AI can use rainfall as a powerful predictor.

The Optimal Rainfall Range for Maize

Low Rainfall 350–600mm Yield reduction of 10–22%
Optimal 600–1200mm Maximum yield potential
Excess Rainfall > 1200mm Flooding/waterlogging damage; yield loss of 21–35% if flooding occurs at critical stages

3. How to Model This in Your AI (Implementation for Replit)

3.1 Rainfall Impact Functions

```python
def calculate_rainfall_impact(rainfall_mm, crop_type, growth_stage):
    """
    Returns a yield adjustment factor based on rainfall.
    
    For maize:
    - Optimal: 600-1200mm
    - Drought: <500mm (severe losses)
    - Excess: >1200mm (flooding risk)
    """
    if crop_type != "maize":
        # Use crop-specific thresholds
        return 1.0
    
    # Optimal range
    if 600 <= rainfall_mm <= 1200:
        return 1.0  # No adjustment
    
    # Drought stress
    if rainfall_mm < 500:
        # Yield loss increases as rainfall drops
        loss_factor = (500 - rainfall_mm) / 500
        # Cap at 100% loss
        return max(0, 1 - loss_factor)
    
    # Moderate low rainfall
    if 500 <= rainfall_mm < 600:
        # 10-22% reduction[reference:15]
        reduction = 0.10 + (0.12 * (600 - rainfall_mm) / 100)
        return 1 - reduction
    
    # Excess rainfall (flooding risk)
    if rainfall_mm > 1200:
        excess = rainfall_mm - 1200
        # Base reduction increases with excess
        base_reduction = min(0.35, excess / 3000)  # Cap at 35%
        
        # Growth stage multiplier
        stage_multiplier = {
            "germination": 0.8,
            "vegetative": 0.5,
            "tasseling": 1.0,  # Most sensitive[reference:16]
            "silking": 0.9,
            "grain_filling": 0.6,
            "harvest": 0.7  # Rotting risk[reference:17]
        }
        multiplier = stage_multiplier.get(growth_stage, 0.5)
        
        return 1 - (base_reduction * multiplier)
    
    return 1.0
```

3.2 Updated Yield Prediction with Rainfall

```python
def predict_yield_with_rainfall(base_yield, rainfall_mm, crop, growth_stage, ndvi):
    # Get rainfall adjustment
    rain_factor = calculate_rainfall_impact(rainfall_mm, crop, growth_stage)
    
    # NDVI adjustment (NDVI correlates with rainfall at R²=0.437)[reference:18]
    # NDVI strongly driven by precipitation (r=0.72-0.88)[reference:19]
    ndvi_factor = 0.5 + (ndvi * 0.5)  # Normalize NDVI to 0.5-1.0 range
    
    # Combined adjustment
    total_factor = rain_factor * ndvi_factor
    
    return base_yield * total_factor
```

3.3 Dynamic Risk Score with Rainfall

```python
def calculate_risk_score_with_rainfall(base_score, rainfall_mm, crop):
    # Start with base score
    score = base_score
    
    # Drought penalty
    if rainfall_mm < 500:
        # Severe drought = high risk
        drought_penalty = (500 - rainfall_mm) / 100  # Up to 5 points
        score -= drought_penalty
    
    # Excess rainfall penalty
    if rainfall_mm > 1200:
        flood_penalty = (rainfall_mm - 1200) / 300  # Up to 3 points
        score -= flood_penalty
    
    # Clamp to 1-10
    return max(1, min(10, score))
```
