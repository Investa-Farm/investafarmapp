🛰️ Part 1: Satellite Monitoring with Sentinel-2
We'll use the free and open data from the European Space Agency's Sentinel-2 satellites.

1. Create a Copernicus Open Access Hub Account
This is your key to the data.

Go to the Copernicus Open Access Hub.

Click "Sign Up" and register a new account.

Once your account is approved, take note of your username and password.

2. Install Required Python Libraries
In your Replit environment, run this command to install the tools:

bash
pip install sentinelsat rasterio numpy
3. Write the Sentinel-2 Downloader
Create a new file, download_sentinel.py. The script below finds and downloads the latest Sentinel-2 image for a given GPS coordinate.

python
from sentinelsat import SentinelAPI, read_geojson, geojson_to_wkt
from datetime import date

# --- Setup (Replace with your credentials) ---
api = SentinelAPI('YOUR_COPERNICUS_USERNAME', 'YOUR_COPERNICUS_PASSWORD')
# The GPS coordinates of your farm
farm_gps_point = (36.8219, -1.2921)  # (longitude, latitude) for Nairobi example

# Define the search area (a 1km x 1km box around the GPS point)
longitude, latitude = farm_gps_point
search_area = {
    'type': 'Polygon',
    'coordinates': [[
        [longitude - 0.01, latitude - 0.01],
        [longitude + 0.01, latitude - 0.01],
        [longitude + 0.01, latitude + 0.01],
        [longitude - 0.01, latitude + 0.01],
        [longitude - 0.01, latitude - 0.01]
    ]]
}

# Search for Sentinel-2 images from the last 10 days, with less than 20% cloud cover
products = api.query(
    geojson_to_wkt(search_area),
    date=(date.today().replace(day=1), date.today()), # Get images from the start of the month
    platformname='Sentinel-2',
    cloudcoverpercentage=(0, 20)
)

# If an image is found, download it
if products:
    # Get the first (most recent) product
    product_id = list(products.keys())[0]
    api.download(product_id, './data/')
    print(f"Successfully downloaded image for {farm_gps_point}")
else:
    print(f"No recent cloud-free images found for {farm_gps_point}")
Source: Copernicus Open Access Hub Access and Python API documentation

4. Calculate NDVI (Crop Health Score)
After downloading an image, you need to calculate NDVI. This measures plant health by comparing near-infrared (Band 8) and red light (Band 4). The next script, calculate_ndvi.py, processes the downloaded data.

python
import rasterio
import numpy as np

# Use the path to the downloaded Sentinel-2 .jp2 file
# Replace with your actual file path
band_red_path = './data/S2A_MSIL1C_20230601T.../GRANULE/.../IMG_DATA/R10m/T36QWE_20230601T..._B04_10m.jp2'
band_nir_path = './data/S2A_MSIL1C_20230601T.../GRANULE/.../IMG_DATA/R10m/T36QWE_20230601T..._B08_10m.jp2'

with rasterio.open(band_red_path) as src_red:
    red = src_red.read(1).astype(float)

with rasterio.open(band_nir_path) as src_nir:
    nir = src_nir.read(1).astype(float)

# Calculate NDVI
np.seterr(divide='ignore', invalid='ignore')
ndvi = (nir - red) / (nir + red)

# Calculate average NDVI for the farm (ignore NaN values)
average_ndvi = np.nanmean(ndvi)

# The NDVI ranges from -1 to 1.
# A healthy crop is typically above 0.3.
print(f"The average NDVI for the farm is: {average_ndvi}")
Source: Sentinel Hub NDVI Calculation Examples

⛅ Part 2: Weather Monitoring with Open-Meteo
Open-Meteo is a fantastic, completely free, open-source weather API with no API keys required.

1. Install the Client Library
In your Replit environment, run:

bash
pip install openmeteo-requests
2. Write the Weather Fetcher
Create a file named get_weather.py. It will pull real-time and forecast data for any GPS coordinate.

python
import openmeteo_requests
import pandas as pd

# Setup the Open-Meteo API client
openmeteo = openmeteo_requests.Client()

# The GPS coordinates for your farm
farm_gps_point = {"latitude": -1.2921, "longitude": 36.8219}  # Nairobi example

# Define the weather variables you need
params = {
    "latitude": farm_gps_point["latitude"],
    "longitude": farm_gps_point["longitude"],
    "hourly": ["temperature_2m", "rain", "soil_moisture_0_to_10cm"],
    "daily": ["temperature_2m_max", "temperature_2m_min", "precipitation_sum"],
    "timezone": "Africa/Nairobi"
}

# Fetch the data from the Open-Meteo API
responses = openmeteo.weather_api("https://api.open-meteo.com/v1/forecast", params=params)

# Process the first response
response = responses[0]
print(f"Weather report for: {response.Latitude()}°N, {response.Longitude()}°E")

# Get the forecast data
daily = response.Daily()
daily_data = {
    "date": pd.date_range(
        start=pd.to_datetime(daily.Time(), unit="s", utc=True),
        periods=len(daily.Variables(0).ValuesAsNumpy()),
        freq="D"
    ),
    "max_temp_c": daily.Variables(0).ValuesAsNumpy(),
    "min_temp_c": daily.Variables(1).ValuesAsNumpy(),
    "rain_sum_mm": daily.Variables(2).ValuesAsNumpy(),
}

daily_df = pd.DataFrame(data=daily_data)
print("\n5-Day Weather Forecast:")
print(daily_df.head())
