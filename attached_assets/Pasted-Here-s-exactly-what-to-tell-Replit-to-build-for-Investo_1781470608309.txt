Feature: AI‑Powered Portfolio Builder for Qualified Investors

Qualification Trigger

· When an investor’s total invested amount (across all farms, past and present) reaches ≥ $5,000 USD (or KES equivalent), they automatically qualify as a Portfolio Manager.
· A pop‑up notification appears on their dashboard:
  “🎉 You’re now a Portfolio Manager! Build & share your own farm portfolios using AI. Click here to start.”
· Qualification is realtime – if their invested amount drops below $5,000 (e.g., after selling), they lose the ability to create new portfolios but existing portfolios remain active.

Portfolio Manager Onboarding Pop‑Up (First Time)

After clicking, show a three‑step wizard:

1. Name your portfolio (e.g., “Mega Maize Growth”)
2. Select your strategy (AI will pre‑fill based on this choice)
   · High Growth → more maize, higher risk
   · Balanced → mix of crops + regions
   · Climate Resilient → farms with high NDVI & low weather risk
   · Custom → manual override
3. Set target risk level (slider 1–10)

AI Portfolio Builder (Core Logic)

The AI does the following automatically, but the portfolio manager can edit any value:

· Farm selection: Scans all currently active farms (primary or secondary available) that match the chosen strategy and risk level.
· Weight allocation: Uses Modern Portfolio Theory to optimise for expected return given risk constraint.
  Output: list of farms + percentage weight (sum 100%).
· Display to manager: Show table with farm name, crop, region, NDVI score, risk score, expected return %, and assigned weight.
· Manual override: Manager can add/remove farms, adjust weights, rename portfolio.

Under the hood (to tell Replit):

Build an AI optimisation function that takes:

· Target risk level (1–10)
· Strategy keyword (growth/balanced/resilient)
· List of all investable farms with their: expected return (from forecast), risk score (1–10), crop type, region, NDVI

The function solves for weights that maximise expected return while staying within a risk budget derived from target risk level. Use a simple mean‑variance optimisation (e.g., using scipy.optimize). For MVP, do not require covariance matrix – use heuristic: reduce allocation to farms with risk score > (target+2) and increase allocation to farms with risk score < target.

Sharing & Distribution

After building, the portfolio manager can:

· Publish → portfolio appears in a “Community Portfolios” section on the platform.
· Share link → generate a unique URL (e.g., investafarm.com/portfolio/mega-maize-123) that any investor can open.
· Set fee → optional management fee (0% – 2% per year) that the portfolio manager earns from followers. Fee deducted from investor’s harvest earnings before payout.

Other investors can:

· Click “Copy this portfolio” – automatically buys all underlying farms with the same weights, using their own funds.
· Choose “Auto‑rebalance” – if the manager updates weights, followers get a notification and option to rebalance (or auto‑rebalance if they enabled it).

API Endpoints to Build

Endpoint Method Description
/portfolio/qualification/{investor_id} GET Returns {qualified: true, total_invested: 5200}
/portfolio/generate-ai POST Accepts {target_risk, strategy} → returns suggested holdings
/portfolio/create POST Saves portfolio name, holdings, fee
/portfolio/{id}/publish POST Makes portfolio public
/portfolios/community GET Lists published portfolios (paginated, sorted by follower count)
/portfolio/{id}/copy POST Creates a subscription for the calling investor, executes all buy orders

Fee & Reward Logic

· The portfolio manager earns management fee = (investor’s total value in that portfolio) × (annual fee / 365) deducted daily.
· Platform takes 30% of management fee as commission; 70% goes to portfolio manager.
· Fees are paid from investor’s wallet each day (if insufficient, accrued).

Important Rules

· A portfolio manager cannot include their own farms (no self‑dealing) unless the farm is open to all investors on equal terms.
· Portfolios are rebalanced only when the manager explicitly edits and republishes. Followers get an in‑app notification: “Mega Maize Growth has been rebalanced. View changes & update your copy?”
· If a farm in a portfolio becomes fully subscribed or is no longer available, the AI automatically suggests a replacement when the manager edits.
