7. Complete Algorithm for Replit (Primary Market)
Here’s the step‑by‑step calculation you can implement:

python
# Primary market pricing for Investa Farm
# Given inputs
K = 15000                # dollars, farmer's capital need
R_hat = 100000           # forecast revenue (example)
T = 120                  # days to harvest
r_f = 0.10               # annual risk-free rate
S = 7                    # AI risk score (1-10)
p_max = 0.4
L = 0.8

# Calculate risk discount
loss_prob = ((10 - S) / 9.0) * p_max
risk_discount = loss_prob * L

# Time discount
time_discount = (1 + r_f) ** (T / 365.0)

# Combined discount factor d
d = (1 - risk_discount) / time_discount

# Required investor share alpha_req (if K is fixed)
alpha_req = K / (R_hat * d)

# Check viability
if 0.10 <= alpha_req <= 0.30:
    print(f"Viable: alpha = {alpha_req:.2%}")
else:
    print(f"Not viable: alpha = {alpha_req:.2%} (outside 10-30%)")

# Choose N (e.g., 1500 for nice $10 shares)
N = 1500
P0 = K / N

print(f"Primary share price: ${P0:.2f}")
print(f"Number of shares: {N}")
print(f"Investor revenue share required: {alpha_req:.2%}")
