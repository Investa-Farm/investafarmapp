1. Database Schema (PostgreSQL / SQLite)

Table: farms

Stores static farm information and current season parameters.

Column Type Description
id SERIAL PRIMARY KEY Unique farm ID
name VARCHAR e.g., "Kilimo Bora Maize"
location VARCHAR Region, GPS coordinates
crop_type VARCHAR e.g., "maize", "beans"
season_start DATE Planting date
season_end DATE Expected harvest date
capital_needed DECIMAL(12,2) K in USD (e.g., 15000.00)
forecast_revenue DECIMAL(12,2) Updated daily by AI
farmer_share DECIMAL(5,4) β (0.45–0.65)
investor_share DECIMAL(5,4) α (0.10–0.30) – can be computed or stored
platform_share DECIMAL(5,4) γ = 1-α-β
total_shares INTEGER N
share_price_primary DECIMAL(10,2) P0 in USD
risk_score SMALLINT 1–10, updated daily by AI
p_max DECIMAL(5,4) 0.4 (default, can vary by crop)
loss_given_default DECIMAL(5,4) 0.8
risk_free_rate DECIMAL(5,4) 0.10

Table: share_orders

Tracks secondary market bids and asks (order book).

Column Type Description
id SERIAL PRIMARY KEY 
farm_id INTEGER REFERENCES farms(id) 
order_type VARCHAR 'buy' or 'sell'
quantity INTEGER number of shares
limit_price DECIMAL(10,2) USD per share (optional, if market order use formula)
status VARCHAR 'open', 'filled', 'cancelled'
created_at TIMESTAMP 

Table: price_history

Logs daily fair values for audit and charting.

Column Type Description
id SERIAL PRIMARY KEY 
farm_id INTEGER REFERENCES farms(id) 
date DATE 
fair_value_secondary DECIMAL(10,2) Computed seller price (without fees)
closing_price DECIMAL(10,2) Last executed trade price
volume INTEGER Number of shares traded

2. API Endpoint Design (REST)

Base URL: https://api.investafarm.com/v1

GET /farms/{farm_id}/primary-price

Response (example for farm with capital_needed=15000, total_shares=1500):

2.3 Place Secondary Market Order

POST /orders

Request body:

```json
{
  "farm_id": 101,
  "order_type": "sell",
  "quantity": 100,
  "limit_price": null   // null means market order (use current formula)
}
```

Response:

```json
{
  "order_id": 5001,
  "status": "open",
  "estimated_execution_price": 8.35,
  "fee": 0.0835
}
```

2.4 Get No‑Arbitrage Check

GET /farms/{farm_id}/arbitrage-check

Compares current market price to fair value.

Response:

```json
{
  "fair_value_no_arb": 9.12,
  "current_market_price_seller": 8.35,
  "deviation_percent": -8.44,
  "arbitrage_opportunity": false,
  "alert": "Price below fair value – buying opportunity"
}
```

3. Implementation in Replit (Python + Flask)

Here’s a complete, runnable code skeleton you can paste directly into a Replit Python project.

```python
from flask import Flask, request, jsonify
from datetime import date, timedelta
import math

app = Flask(__name__)

# ---------- Pricing Functions ----------
def time_discount_factor(days, rate=0.10):
    return (1 + rate) ** (days / 365.0)

def risk_discount(risk_score, p_max=0.4, loss_given_default=0.8):
    loss_prob = ((10 - risk_score) / 9.0) * p_max
    return loss_prob * loss_given_default

def secondary_seller_price(alpha, forecast_revenue, total_shares, days_to_harvest,
                           risk_score, buy_orders, sell_orders, liquidity_discount=0.02,
                           beta_sens=0.2, r_f=0.10, p_max=0.4, L=0.8):
    # Expected dividend
    expected_div = (alpha * forecast_revenue) / total_shares
    # Time discount
    time_factor = time_discount_factor(days_to_harvest, r_f)
    price_time = expected_div / time_factor
    # Risk discount
    rd = risk_discount(risk_score, p_max, L)
    price_risk = price_time * (1 - rd)
    # Demand/supply imbalance
    total = total_shares
    if total > 0:
        imbalance = (buy_orders - sell_orders) / total
    else:
        imbalance = 0
    demand_mult = 1 + beta_sens * imbalance
    price_demand = price_risk * demand_mult
    # Seller price after liquidity discount
    seller = price_demand * (1 - liquidity_discount)
    return seller, price_risk, price_time, demand_mult

def primary_alpha_required(K, forecast_revenue, days_to_harvest, risk_score,
                           r_f=0.10, p_max=0.4, L=0.8):
    rd = risk_discount(risk_score, p_max, L)
    time_factor = time_discount_factor(days_to_harvest, r_f)
    d = (1 - rd) / time_factor
    alpha_req = K / (forecast_revenue * d)
    return alpha_req

# ---------- Mock Data ----------
farms_db = {
    101: {
        "name": "Kilimo Bora Maize",
        "capital_needed": 15000.0,
        "total_shares": 1500,
        "investor_share": 0.20,   # alpha
        "forecast_revenue": 100000.0,
        "risk_score": 7,
        "days_to_harvest": 90,
        "p_max": 0.4,
        "loss_given_default": 0.8,
        "risk_free_rate": 0.10,
    }
}

# ---------- API Endpoints ----------
@app.route('/v1/farms/<int:farm_id>/primary-price', methods=['GET'])
def primary_price(farm_id):
    farm = farms_db.get(farm_id)
    if not farm:
        return jsonify({"error": "Farm not found"}), 404
    
    K = farm["capital_needed"]
    N = farm["total_shares"]
    P0 = K / N
    
    # Check consistency
    alpha_req = primary_alpha_required(
        K, farm["forecast_revenue"], farm["days_to_harvest"],
        farm["risk_score"], farm["risk_free_rate"],
        farm["p_max"], farm["loss_given_default"]
    )
    viable = 0.10 <= alpha_req <= 0.30
    
    return jsonify({
        "farm_id": farm_id,
        "share_price": round(P0, 2),
        "currency": "USD",
        "total_shares": N,
        "investor_share_alpha": farm["investor_share"],
        "forecast_revenue": farm["forecast_revenue"],
        "risk_score": farm["risk_score"],
        "days_to_harvest": farm["days_to_harvest"],
        "fair_value_check": {
            "alpha_required_given_K": round(alpha_req, 4),
            "status": "viable" if viable else "adjust alpha or capital"
        }
    })

@app.route('/v1/farms/<int:farm_id>/secondary-price', methods=['GET'])
def secondary_price(farm_id):
    farm = farms_db.get(farm_id)
    if not farm:
        return jsonify({"error": "Farm not found"}), 404
    
    # For demo, mock order book values
    buy_orders = 2000
    sell_orders = 1000
    
    seller_price, price_risk, price_time, demand_mult = secondary_seller_price(
        alpha=farm["investor_share"],
        forecast_revenue=farm["forecast_revenue"],
        total_shares=farm["total_shares"],
        days_to_harvest=farm["days_to_harvest"],
        risk_score=farm["risk_score"],
        buy_orders=buy_orders,
        sell_orders=sell_orders,
        liquidity_discount=0.02,
        beta_sens=0.2,
        r_f=farm["risk_free_rate"],
        p_max=farm["p_max"],
        L=farm["loss_given_default"]
    )
    
    trading_fee = 0.01
    buyer_price = seller_price / (1 - 0.02) * (1 + trading_fee)  # simplified
    
    return jsonify({
        "farm_id": farm_id,
        "seller_price": round(seller_price, 2),
        "buyer_price": round(buyer_price, 2),
        "trading_fee": trading_fee,
        "mid_price": round((seller_price + buyer_price)/2, 2),
        "order_book_summary": {
            "total_bid_quantity": buy_orders,
            "total_ask_quantity": sell_orders,
            "imbalance_factor": round(demand_mult, 4)
        },
        "breakdown": {
            "expected_dividend": round((farm["investor_share"] * farm["forecast_revenue"]) / farm["total_shares"], 2),
            "time_discount": round(time_discount_factor(farm["days_to_harvest"], farm["risk_free_rate"]), 4),
            "risk_discount": round(risk_discount(farm["risk_score"], farm["p_max"], farm["loss_given_default"]), 4),
            "liquidity_discount": 0.02,
            "demand_multiplier": round(demand_mult, 4)
        },
        "updated_at": date.today().isoformat()
    })

if __name__ == '__main__':
    app.run(host='0.0.0.0', port=5000)
```
